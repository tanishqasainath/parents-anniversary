A beautiful, single-page anniversary website — built with love, designed with care. Warm gold tones, elegant typography, scroll-triggered animations, and a heartfelt letter section make this the perfect way to celebrate your parents' journey together.

## ✨ Features

- **Timeless design** — Warm neutral palette with gold accents, serif & sans-serif pairings
- **Scroll animations** — Elements fade and reveal as you scroll
- **Timeline** — Chronicle their journey year by year
- **Photo gallery** — Grid layout ready for your favourite pictures
- **Love letter** — A dedicated space for your own words from the heart
- **Fully responsive** — Looks beautiful on any device

## 🎨 Preview

The site includes these sections:

| Section | Description |
|---|---|
| **Hero** | Anniversary title, names, date — full-screen welcome |
| **Message** | A short, framed greeting |
| **Timeline** | Key milestones in their journey |
| **Gallery** | A 3-column photo grid (replace placeholders with real photos) |
| **Love Letter** | Your own heartfelt letter to them |
| **Closing** | Years/days count with a final wish |

## 🛠️ Customisation

Open `index.html` and replace every `[Placeholder]`:

| Placeholder | Replace with |
|---|---|
| `[Dad's Name]` & `[Mom's Name]` | Your parents' names |
| `[X Years]` | Number of years married |
| `[Month DD, YYYY]` | Anniversary date |
| `[Year]` | Year of each milestone |
| `[Milestone]` | Event label (e.g. "The Day We Met") |
| `[Description]` | A short memory for each milestone |
| `[Photo 1–6]` | Replace with `<img src="photo.jpg" alt="...">` tags |
| `[Your Name]` | Your name (or the sender) |
| **Letter section** | Write your own message from the heart |
