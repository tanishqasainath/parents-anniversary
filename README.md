# Parents Anniversary Website

A heartfelt anniversary website for **[Dad's Name]** and **[Mom's Name]** — celebrating **[X Years]** of love.

## 🖼️ How to customise

Open **`index.html`** and replace all `[Placeholder]` text:

- `[Dad's Name]` / `[Mom's Name]` — your parents' names
- `[X Years]` — number of years married
- `[Month DD, YYYY]` — anniversary date
- `[Year]` / `[Milestone]` / `[Desc]` — timeline entries
- `[Photo 1–6]` — replace the gallery placeholder divs with actual `<img>` tags
- `[Your Name]` — your name
- The **letter section** — write your own heartfelt message

## 🚀 Deploy to GitHub Pages

```bash
# push to GitHub
git remote add origin <your-repo-url>
git push -u origin main

# then go to Settings → Pages → select "main" branch
```
